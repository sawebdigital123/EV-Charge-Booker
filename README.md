The EV Charge Booker is a dedicated web application designed to solve the problem of range anxiety and long wait times at electric vehicle (EV) charging stations. It provides EV owners with a seamless, real-time platform to locate nearby charging stations, check slot availability, and book a charging session in advance.

This system aims to optimize charging infrastructure usage, enhance the convenience of EV ownership, and contribute to the broader adoption of sustainable transportation.

‚ú® Key Features

The application is built around user convenience and efficient resource management.

User Experience (UX)

Station Locator: An interactive map view powered by a mapping API (e.g., Google Maps) to visually locate nearby charging stations.

Real-Time Availability: Display of the current status of charging ports (Available, Occupied, Out of Service).

Advance Slot Booking: Users can select a date, time, and charging duration to reserve a specific slot.

Filtering & Search: Options to filter stations by Connector Type (e.g., CCS, CHAdeMO, Type 2), Charging Speed (e.g., DC Fast Charge, Level 2), and Price.

User Dashboard: A personalized space to view upcoming bookings, charging history, and manage vehicle details.

Technical & System Features

Secure Authentication: User registration and login powered by Firebase Authentication.

Real-Time Database: Utilizes Firestore to ensure immediate synchronization of booking data and station status across all users and devices.

Responsive Design: Fully adaptive UI built with Tailwind CSS, providing a consistent experience on desktop, tablet, and mobile devices.
